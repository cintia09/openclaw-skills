# 新高考I卷数学考试规范（2024-2025年）

## 一、基本信息

- **适用范围**：江苏省高考（使用全国新高考I卷）
- **考试时间**：120分钟
- **卷面满分**：150分
- **考试形式**：闭卷、笔试

## 二、试卷结构（8+3+3+5模式）

### 1. 单项选择题（第1-8题）
- **题量**：8题
- **分值**：每题5分，共40分
- **要求**：在四个选项中选出唯一正确答案

### 2. 多项选择题（第9-11题）
- **题量**：3题
- **分值**：每题6分，共18分
- **评分规则**：
  - 全部选对得6分
  - 部分选对得3分
  - 有选错或不选得0分

### 3. 填空题（第12-14题）
- **题量**：3题
- **分值**：每题5分，共15分

### 4. 解答题（第15-19题）
- **题量**：5题
- **分值分布**：13分 + 15分 + 15分 + 17分 + 17分 = 77分
- **具体分配**：
  - 第15题：13分
  - 第16题：15分
  - 第17题：15分
  - 第18题：17分
  - 第19题：17分（压轴题）

## 四、难度梯度设置

- **基础题**：约50%（75分）
  - 单选1-4、多选9、填空12、解答15
- **中等题**：约30%（45分）
  - 单选5-7、多选10、填空13、解答16-17
- **较难题**：约20%（30分）
  - 单选8、多选11、填空14、解答18-19

## 五、注意事项

1. **多选题评分**：部分正确得3分，这是与单选题的重要区别
2. **解答题要求**：必须写出完整的解题过程，仅有答案不得分
3. **知识模块覆盖**：
   - 必考：集合、复数、向量、三角、立体几何、概率统计、解析几何、导数
   - 轮考：数列、函数综合、排列组合（从中选取部分考查）
4. **计算器规定**：不允许使用计算器

## 六、LaTeX公式书写规范

- 行内公式：`$...$`
- 行间公式：`$$...$$`
- 绝对值：`\lvert x \rvert`（不用 `|x|`）
- 集合中"使得"：`\mid`（不用 `|`）
- 分数：`\frac{a}{b}`
- 根号：`\sqrt{x}` 或 `\sqrt[n]{x}`
