# 医院门诊查询（hospital-lookup）

> 目标：帮用户找到**当前可以挂号**的医院和科室，不是给一堆搜索结果让用户自己打电话确认。

## 1. 触发场景

- "XX区哪个医院有XX科门诊？"
- "今天/周末能去哪里看XX科？"
- "帮我查一下XX医院XX科的出诊时间"
- "哪个医院的XX科比较好？能挂号吗？"

## 2. 核心原则：亲自验证，不靠搜索猜测

**❌ 错误做法：**
- 搜索"XX区 XX科 医院"→ 把搜索结果列出来 → 说"建议电话确认"
- 这等于没帮忙，用户自己也能搜

**✅ 正确做法：**
1. 先搜索找到该区域有XX科的医院列表
2. **逐一访问每家医院的官网**，查看：
   - 门诊时间表（周几有门诊）
   - 专家排班表（具体哪个医生哪天出诊）
   - 挂号须知（是否支持当日挂号、线上挂号渠道）
3. 如果官网信息不全，尝试：
   - 医院微信公众号（搜索公众号名称）
   - 南京12320预约平台：https://www.nj12320.org
   - 好大夫在线：https://www.haodf.com
   - 挂号网/微医：https://www.guahao.com
4. 汇总后给出**明确结论**：今天/明天能不能去，有没有号

## 3. 输出格式

对每家医院，给出：

```
医院名称 ✅/❌ 当日可挂号/不可挂号
📍 地址
⏰ 门诊时间：周X-周X，上午X:XX-X:XX，下午X:XX-X:XX
👨‍⚕️ 今日出诊：XX医生（主任医师）/ 普通门诊
📞 电话：XXX（如果关键信息缺失，附电话让用户确认）
🔗 信息来源：官网URL
```

## 4. 信息不确定时的处理

- 官网明确写了门诊时间 → 直接给结论
- 官网没写具体排班 → 标注"官网未列排班，建议电话确认：XXX"
- 官网打不开 → 标注"官网无法访问"，尝试其他渠道
- **绝不编造门诊时间**

## 5. 常用查询渠道

### 南京地区
- 南京12320预约挂号：https://www.nj12320.org
- 各医院微信公众号（通常有在线挂号功能）
- 好大夫在线（有详细的医生出诊信息）

### 通用
- 医院官网（首选，最权威）
- 好大夫在线：https://www.haodf.com
- 微医/挂号网：https://www.guahao.com
- 丁香园医院库：https://y.dxy.cn
