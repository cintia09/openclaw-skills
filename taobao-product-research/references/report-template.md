# 报告模板

## 飞书精简版

```
🛒 {商品类目} 真实评价调研报告

1️⃣ {商品名} ¥{价格} ⭐推荐
好评率{X}% | {关键评价摘要}
👉 https://detail.tmall.com/item.htm?id={itemId}

2️⃣ {商品名} ¥{价格}
好评率{X}% | {关键评价摘要}
👉 https://detail.tmall.com/item.htm?id={itemId}

...

❌ 淘汰：{商品名} — {淘汰理由}

💡 建议
{场景A} → {推荐商品}
{场景B} → {推荐商品}
```

## 详细版（聊天回复）

每款商品包含：
- 排名 + emoji奖牌（🥇🥈🥉）
- 价格 + 是否秒杀价
- 好评率 + 评价数量
- 评价关键词标签
- 3-5条真实买家评价摘录
- ⚠️ 注意事项（差评要点）
- 产品认证信息
- 最终判断（一句话总结）

末尾给出：
- ❌ 淘汰商品 + 理由
- 💡 按场景的购买建议
- 🛒 已加购提示
